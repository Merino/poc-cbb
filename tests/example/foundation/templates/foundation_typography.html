This is a paragraph, using Arial, with 14px font size and 21px line height.

 12px Within our guidelines the paragraph demonstrates the generic font, color, and vertical rhythm. The paragraph is composed with a line height of 18px. This value is known as baseline, as it sets the distance between each line where the text "sits". The 18px value is our ideal vertical rhythm, but for general purposes we derived from it a grid of 10px to be used through out our design language. Doing so establishes a consistent tone for the user interface and increases the probability of elements sharing the same alignment/baseline even if set within different columns and/or contexts.
